Add many notes

You may create several notes at once. One note name per line.

To assocate the note item with one or more projects start a line with "Project:". Then list a comma separated list of projects to be associated with the subsequent notes.

For example:
Project: Fed Ex, Smart Drive
New Note 1
New Note 2

You may also specify the type for each new note. For example:

Project: Fed Ex
New Note 1:Numeric

Valid types: Text, Numeric, Alphanumeric, Numeric_with_spaces, Multiline
